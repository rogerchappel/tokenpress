# Roadmap

## V1: sharp little log press

- Deterministic CLI and library API.
- Markdown/JSON output.
- Fixture-backed OpenClaw/Codex/plain heuristics.
- Local-only safety model and default redaction.

## V1.x: better handoffs

- Configurable evidence scoring profiles.
- More fixture formats from real agent/CI transcripts.
- GitHub Actions annotations for checked-in fixture reports.
- Golden snapshot tests for report stability.

## Later, maybe

- HTML reports.
- TUI review mode.
- Optional tokenizer integration for budget estimates.

Non-goal: becoming a hosted log ingestion platform. TokenPress should stay small, local, and easy to reason about.
